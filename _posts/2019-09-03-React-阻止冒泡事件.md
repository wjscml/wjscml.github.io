# 合成事件与原生事件的区别

#### 1、合成事件
在jsx中直接绑定的事件：

    <a ref="aaa" onClick={(e)=>this.handleClick(e)}>更新</a>

#### 2、原生事件

通过js原生代码绑定的事件：

    document.body.addEventListener('click',e => {
        // 通过e.target判断阻止冒泡
        if (e.target&&e.target.matches('a')) {
            return;
        }
        console.log('body');
    })
或
    this.refs.update.addEventListener('click', e => {
        console.log('update');
    });
